Tracing a settlement across 14 services without re-instrumenting anything.

Tessera's settlement pipeline touches authorization, fraud scoring, currency conversion, ledger writes and three downstream banking partners. When a payment took longer than expected, isolating the culprit meant pulling logs from 14 dashboards and reconciling timestamps by hand.

The team installed Riverbed's open-source agent across their Go and Rust services in a single afternoon — no manual span calls, no SDK rollout, no protobuf wrangling. By the end of the week the platform team could click a single payment ID and watch the request flow across every service, queue and database it touched.

In our first month with Riverbed we found and fixed three latency regressions that had been written off as "the network." None of them were the network.

Six months in, Tessera has retired four standalone observability tools, reduced their on-call burden by an estimated 11 hours per engineer per month, and uses Riverbed's query plane during architecture reviews — not just incidents.

Following a pick-task from a tablet, through ROS, to a robot arm — in one trace.

Kasai's pick orchestration spans cloud services, regional edge gateways and ROS 2 nodes running on the robots themselves. Previous APM tools stopped at the cloud boundary, leaving the team to guess whether a 600ms delay came from the dispatcher or the arm controller.

Because Riverbed's agent ships as a single static binary with a Python and C++ shim, the team extended tracing all the way onto the robot — without a vendor SDK and without sending data outside the warehouse's local subnet. The hosted query plane stitches edge spans into the same timeline as cloud spans.

For the first time we can see a robot's behavior and the dispatcher's intent in the same view. It changed what kinds of bugs we even bother filing.

Kasai now uses Riverbed for fleet-wide performance audits, catching arm-calibration drift via span-duration histograms before it shows up as a missed SLA. The team estimates Riverbed has paid for itself twice over in avoided re-picks.
