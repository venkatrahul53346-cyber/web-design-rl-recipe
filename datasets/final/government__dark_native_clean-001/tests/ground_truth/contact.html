Section 14 · Public Enquiries

Contact the Archive.

Reach a reference archivist, request copies of held materials, or notify our offices of access requirements. For document requests covered by the Records Access Act, please file a formal request rather than using this form.

When to use this form
  • General enquiries — finding aids, collection scope, citation help
  • Reference questions — research direction from a duty archivist
  • Press & media — interview requests, embargoed material
  • Accessibility — reading-room accommodations, assistive formats

Standard response time is three working days. Urgent press enquiries are routed to the duty Communications Officer between 09:00 and 18:00.

Records Access Act requests

Requests for unpublished or restricted holdings must be filed through the formal request portal — not via this form. Use Forms & Requests → Records request (RAA). Statutory response window is 20 working days.
