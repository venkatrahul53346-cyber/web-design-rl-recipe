02Review engine
Static · Semantic · Runtime hooks

What the second pair of eyes actually looks at.

Three passes per PR: a structural read of the diff in graph form, a semantic read against the rest of your codebase, and an optional runtime trace replay if you give us OpenTelemetry.

Trigger Webhook on pull_request.opened, .synchronize, and .ready_for_review. Median time-to-first-comment is 38 seconds from webhook receipt.A re-run is queued automatically on force-push; cached embeddings make the second pass ~6× faster.
Languages — tier 1 14TypeScript, JavaScript, Python, Go, Rust, Java, Kotlin, Swift, Ruby, C, C++, C#, PHP, Elixir.Tier 1 means: AST-level diff parsing, type-aware suggestions, and language-specific lint rule promotion.
Languages — tier 2 9Scala, Clojure, Haskell, OCaml, Lua, Dart, Zig, Nim, Crystal — diff-level review only, no AST grounding.
Bug classes detected Null-deref, off-by-one, race conditions on shared state, unhandled promise rejection, SQL injection, XSS via interpolation, missing auth checks on new routes, broken pagination, N+1 queries, secrets in code, leaked PII in logs, dependency CVE introductions.Maintained as a versioned rulebook at github.com/pliny-labs/rulebook — public, MIT.
False-positive rate 3.8%across the 1,840-PR benchmark, measured as comments dismissed by reviewer with the “not useful” reason. We target ≤5% and ship a regression to staging at >6%.
Codebase grounding Incremental indexing on push. Symbols, call graph, and ownership extracted once per repo and updated per commit (median delta: 240ms for a 50-file change).Index lives on your side if self-hosted; otherwise encrypted at rest in the region you choose.
Output format Inline review comments with file + line anchors, an optional summary comment, a structured JSON sidecar for downstream tooling, and webhooks on review.completed.

05Security
Data · Access · Compliance

Where your code lives, and what we promise to do with it.

Default posture is zero-retention. Code is read, embeddings are computed, the review is posted, the buffer is wiped. Long-term storage is opt-in and per-repo.

Data handling DEFAULT

  • Zero-retention mode on by default — diffs purged within 60 seconds of review completion.
  • Embeddings stored encrypted at rest with AES-256, customer-scoped keys.
  • No code is used for training. Ever. Contractual.
  • Regions: us-east-1, eu-west-1, ap-south-1. Pinned per workspace.
  • PII redaction pass before any prompt leaves the perimeter.

Access & auth SSO

  • SAML 2.0 SSO with Okta, Azure AD, Google Workspace, JumpCloud.
  • SCIM 2.0 provisioning and de-provisioning.
  • Scoped tokens with repo-level granularity; no org-wide reads.
  • Audit log streamed to your SIEM via webhook or S3 sink.
  • IP allow-listing on every API surface.

Compliance IN AUDIT

  • SOC 2 Type II — currently in audit window, report Q2 2025.
  • ISO 27001 — readiness assessment complete.
  • GDPR & UK GDPR — DPA available, sub-processor list public.
  • HIPAA — supported on Enterprise plan with signed BAA.
  • Penetration test — annual, Latacora; latest report on request under NDA.
06Deployment
Hosted · VPC · Air-gapped

Three ways to run it. Pick the one your security team won't argue with.

Hosted (default) Multi-tenant on AWS, region-pinned. No infra on your side. Ships same day you sign up. Recommended unless you already operate your own ML infra.SLA 99.9% · transit in TLS 1.3 · keys managed in AWS KMS with HSM backing.
VPC-isolated Single-tenant control plane in your AWS, GCP, or Azure account. We deploy via Terraform; you hold the keys. Models call out to a customer-owned NAT gateway or stay fully internal if you bring your own.Onboarding: ~3 business days. Requires a 2-vCPU control node and a model backend of your choice.
Air-gapped on-prem For regulated industries and defense. Ships as a signed OCI bundle. Includes a local Llama 3.3 70B checkpoint or accepts any OpenAI-compatible endpoint behind your firewall.No phone-home telemetry. Updates delivered via signed artifact. Enterprise plan only.
Hardware recommendation For self-hosted model serving: 2× NVIDIA A100 80GB or 2× H100 80GB. CPU control plane fits on a single c7i.2xlarge / n2-standard-8.Pliny itself is light — the bill is the model. Customers report ~$0.07 per PR amortized on Llama 3.3 70B at 50 PRs/day.
